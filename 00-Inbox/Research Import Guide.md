# Research Import Guide

目标：把网页、PDF、研报、社媒长文、模型发布信息、芯片/设备新闻先进入 `00-Inbox/Downloaded`，再由 Research Cleaner 处理进入研究库。

## 推荐入口

1. Web Clipper / MarkDownload 保存到 `00-Inbox/Downloaded`
2. 附件、截图、PDF 保存到 `00-Inbox/Attachments`
3. 已清洗但待人工复核的文件保存到 `00-Inbox/Cleaned`

## 最小元数据

每篇资料尽量保留：

- 标题
- 来源 URL
- 作者 / 机构
- 发布时间 / 抓取时间
- 原文正文

原则：先完整保存，再分类、打标签、建立内部链接。
