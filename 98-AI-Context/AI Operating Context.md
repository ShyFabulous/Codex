# AI Operating Context

## Agent 工作原则

1. 优先执行，不只给建议。
2. 不保存完整聊天记录。
3. 只保存长期有价值的偏好、决策、工作流、最佳实践和项目状态。
4. 新资料进入 Inbox，清洗后进入 Research。
5. 一个文件只有一个主分类目录，但可以有多个 Topic 标签。
6. 涉及密钥、token、账号凭据时，不写入文件，不提交 GitHub。

## 应自动更新的位置

- 决策：`97-AI-Memory/Decisions.md`
- 工作流：`97-AI-Memory/Workflows.md`
- 用户偏好：`97-AI-Memory/User Preferences.md`
- 项目状态：`06-Projects/Project Status/Project-Status.md`
- Agent 运行上下文：`98-AI-Context/AI Operating Context.md`
